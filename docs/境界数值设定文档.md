修仙世界的境界体系从凡人到渡劫期共分为9个大境界，每个大境界分为前中后期（部分境界有圆满期），共计30个小境界。境界越高，突破难度递增，战力指数级增长。

## 数值设计原则

### 1. 修为需求递增规律
- **炼气期**: 线性增长（100 → 5,000）
- **筑基期**: 4倍跨越（5K → 80K）
- **金丹期**: 12.5倍跨越（80K → 1M）
- **元婴期**: 10倍跨越（1M → 10M）
- **化神期**: 10倍跨越（10M → 100M）
- **炼虚期**: 10倍跨越（100M → 1B）
- **合体期**: 10倍跨越（1B → 10B）
- **大乘期**: 10倍跨越（10B → 100B）
- **渡劫期**: 10倍跨越（100B → 1T）

**规律**: 每个大境界的修为需求呈10倍增长，体现境界越高越难的特点。

### 2. 突破概率递减规律
- **凡人→炼气**: 100% - 80%（新手保护）
- **筑基期**: 60% - 40%（开始有挑战）
- **金丹期**: 30% - 20%（明显困难）
- **元婴期**: 10% - 6%（极难）
- **化神期**: 5% - 3%（天堑）
- **炼虚期**: 2.5% - 1.5%（千里挑一）
- **合体期**: 1.2% - 0.8%（万里挑一）
- **大乘期**: 0.6% - 0.4%（十万里挑一）
- **渡劫期**: 0.3% - 0.1%（百万里挑一）

**规律**: 每提升一个大境界，突破概率约减半，体现境界瓶颈。

### 3. 战力指数增长
- **炼气→筑基**: 5倍增长
- **筑基→金丹**: 5倍增长
- **金丹→元婴**: 2.5倍增长
- **元婴→化神**: 2.5倍增长
- **化神→炼虚**: 2.5倍增长
- **炼虚→合体**: 2.5倍增长
- **合体→大乘**: 2.5倍增长
- **大乘→渡劫**: 2.5倍增长

**规律**: 每个境界战力倍增，确保高境界对低境界的碾压优势。

### 4. 反哺修为递增
反哺修为用于子嗣系统，高境界父母对子女的修为加成更高：
- **炼气期**: 1-5点
- **筑基期**: 20-40点
- **金丹期**: 100-200点
- **元婴期**: 500-1,200点
- **化神期**: 2,000-5,000点
- **炼虚期**: 8,000-18,000点
- **合体期**: 25,000-60,000点
- **大乘期**: 100,000-250,000点
- **渡劫期**: 400,000-1,000,000点

## 游戏平衡考量

### 1. 前期节奏
- **炼气期**: 快速通过（1-2小时）
- **筑基期**: 开始体验成长（3-5小时）
- **金丹期**: 中期核心玩法（10-20小时）

### 2. 中期挑战
- **元婴期**: 需要策略和资源积累（50+小时）
- **化神期**: 需要极大耐心和运气（100+小时）

### 3. 后期目标
- **炼虚期以上**: 超长期目标，给核心玩家追求

### 4. 世界名人池配置
- **金丹后期**: 5名（17%）- 玩家可超越的目标
- **元婴后期**: 5名（16%）- 中期挑战目标
- **化神后期**: 4名（13%）- 高难度目标
- **炼虚后期**: 3名（10%）- 极限挑战
- **合体中期**: 3名（10%）- 传说存在
- **大乘初中后**: 6名（17%）- 顶尖强者
- **渡劫初中后**: 3名（10%）- 修真界巅峰

## 技术实现

### TIERS 数组定义
```javascript
export const TIERS = [
  { name: "凡人", maxExp: 100, feedback: 0, chance: 1.0, desc: "肉体凡胎" },
  { name: "炼气初期", maxExp: 500, feedback: 1, chance: 0.95, desc: "引气入体" },
  // ... 共30个境界
  { name: "渡劫后期", maxExp: 1000000000000, feedback: 1000000, chance: 0.001, desc: "飞升在即" }
];
```

### 战力计算公式
```javascript
战力 = 基础值 × (1 + 灵根加成) × (1 + 资质加成) + 装备加成
```

### 世界NPC演化规则
1. **修为增长**: 每年增长当前境界maxExp的5% × 成长率（境界越高成长率越低）
2. **突破判定**: 修为达到上限后，有2%-10%概率突破（境界越高概率越低）
3. **死亡判定**: 每年有0.5%基础概率（境界越高概率越低）
