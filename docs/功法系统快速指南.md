# 功法系统 - 快速开始指南

## ✨ 核心概念

**功法系统**让你根据子嗣的灵根属性选择最合适的修炼功法，实现"因材施教"。

### 🎯 关键要点

1. **功法等级越高，修炼越快**（黄→玄→地→天）
2. **属性匹配很重要**（火灵根配火功法最强）
3. **天灵根+匹配功法=爆炸成长**（1.5倍契合度）

## 🎮 游戏内操作

### 如何查看子嗣功法？

1. 点击子嗣卡片
2. 在详情页找到 **📖 修炼功法** 区域
3. 查看：
   - 功法名称和等阶
   - 契合度评价（用颜色标记）
   - 实际修炼加成百分比

### 如何更换功法？（待实现）

1. 在功法区域点击 **🔄 更换功法**
2. 选择背包中的功法秘籍
3. 系统自动计算新契合度
4. 确认更换

## 📈 数值计算示例

### 示例：火系天灵根修炼九转控火诀

**基础数据：**
- 资质：85（天灵根）
- 功法：九转控火诀（地阶，+100%）
- 契合度：1.5（完美匹配）

**计算过程：**
```
基础速度 = 10
资质系数 = 85 / 50 = 1.7
灵根倍率 = 2.5（天灵根）
功法加成 = 1 + (1.0 * 1.5) = 2.5
宗门加成 = 1.5
词条加成 = 1.0（假设无）

最终速度 = 10 * 1.7 * 2.5 * 2.5 * 1.5 * 1.0 = 159 点/回合
```

相比之下，同样的子嗣如果只修炼吐纳法：
```
最终速度 = 10 * 1.7 * 2.5 * 1.16 * 1.5 * 1.0 = 74 点/回合
```

**差距：2.15 倍！**

## ⚠️ 常见误区

### 误区1："天阶功法一定最强"

**错误示例：**
- 水灵根修炼 九天引雷术（天阶雷系）
- 实际效果：2.0 * 0.5 = +100%

**正确做法：**
- 水灵根修炼 沧海归心诀（玄阶水系）
- 实际效果：0.5 * 1.0 = +50%（虽然低，但比不匹配的天阶好）

等等，这个算错了！让我重新算：
- 天阶不匹配：+200% * 0.5 = +100%
- 玄阶匹配：+50% * 1.0 = +50%

所以这个例子天阶确实更强。让我重新设计：

### 误区1："只看品阶不看契合度"

**场景：** 你有一本地阶火系功法和一本玄阶水系功法，你的子嗣是水木双灵根。

**错误做法：**
- 选择地阶火系功法（+100% * 0.5 = +50%）

**正确做法：**
- 选择玄阶水系功法（+50% * 1.0 = +50%）

虽然数值相同，但考虑到：
- 玄阶功法可能消耗更少
- 不会有走火入魔风险（待实现）

---

### 误区2："五灵根就是废柴"

**真相：**
- 五灵根如果修炼 **五行至尊经**（天阶，专属）
- 契合度：1.5 倍

**比天灵根修炼普通地阶功法还强！**

## 🔮 进阶技巧

### 1. 宗门选择策略

**根据灵根选宗门：**
- 火灵根 → 丹鼎阁（地阶火功）
- 金灵根 → 青云剑宗（玄阶金功）
- 雷灵根 → 天魔教（天阶雷功）
- 五灵根 → 合欢宗（无属性通用）

### 2. 培养优先级

**S级：** 天灵根 + 对应宗门顶级功法
- 例如：火系天灵根加入丹鼎阁真传

**A级：** 变异灵根 + 对应功法
- 例如：雷灵根 + 大自在天魔经

**B级：** 双灵根 + 匹配功法
- 例如：水木双灵根 + 水系玄阶功法

**C级：** 三灵根以上 + 无属性功法
- 先凑合用，后期换
