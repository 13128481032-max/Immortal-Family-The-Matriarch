# 📰 修真界邸报 - 快速开始

## 一分钟了解邸报系统

### 是什么？
每3个月自动生成的修真界八卦报纸，记录你和NPC的精彩事件。

### 在哪里？
- **查看邸报**：点击左下角 📰 按钮
- **系统设置**：底部导航 → 系统设置 → 修真界邸报

### 两种模式

#### 🤖 AI模式（推荐）
- **需要**：配置DeepSeek API Key
- **优点**：内容生动有趣，像真人写的八卦新闻
- **成本**：约0.001元/次

**配置步骤：**
1. 注册DeepSeek账号：https://platform.deepseek.com/
2. 获取API Key
3. 在系统设置 → AI对话配置中填入
4. 保存并测试连接

#### 📝 模板模式（离线可用）
- **需要**：无需任何配置
- **优点**：完全离线，快速生成
- **缺点**：内容较固定

**使用方法：**
- 不配置API Key即自动使用
- 或在系统设置中关闭"使用AI生成邸报"

### 快速测试
1. 进入游戏后，连续推进时间到第3个月
2. 系统自动弹出第1期邸报
3. 点击左下角📰按钮可随时查看

### 常见内容
- 🎉 **头版头条**：你突破境界、子嗣出生、NPC重大事件
- 💬 **坊间传闻**：次要事件、江湖传闻
- 🏆 **天机榜**：本季修为最高的三位修士
- 📢 **广告位**：修真界趣味广告（彩蛋）

### 往期回顾
- 所有历史邸报都会保存
- 在邸报弹窗中点击"往期回顾"即可查看
- 可以重温你的修仙历程

### 关闭方法
如果不想看邸报：
1. 系统设置 → 修真界邸报
2. 取消勾选"启用修真界邸报"
3. 保存配置

---

**提示**：邸报会在每年的3、6、9、12月自动生成，推进时间时会自动弹出，非常方便！
