# 复仇邸报系统 - 快速开始

## 🚀 5分钟上手

### 1. 打开游戏

启动游戏后，在底部导航栏找到**"复仇"**标签页。

### 2. 查看宿敌状态

复仇界面会显示：
- **境界**: 宿敌当前修为等级
- **状态**: 正常/焦虑/暴怒
- **威胁度**: 0-100%，超过80%极度危险！
- **谣言效果**: 已削弱的战力百分比

### 3. 第一次操作

#### 散布谣言
1. 调整灵石投入（50-150灵石）
2. 点击"散布谣言"按钮
3. 成功后宿敌修为被削弱，威胁度上升
4. **重要**: 每月只能使用一次

#### 查看邸报反馈
下个月打开邸报（地图界面），你会看到：
- 头条新闻："楚清瑶疑似走火入魔"（轻度谣言）
- 或"楚清瑶心魔缠身，境界跌落"（重度谣言）

### 4. 威胁度管理

当威胁度超过30%时：
- 邸报会出现："楚家发布悬赏令"

当威胁度超过80%时：
- 界面出现红色警告
- 邸报头条："血榜杀手出动！"
- **必须立即隐匿行踪！**

#### 隐匿行踪
1. 选择隐匿月数（1-3月）
2. 点击"隐匿行踪"
3. 威胁度大幅降低
4. 期间不会触发刺杀事件

### 5. 准备决斗

推荐条件：
- ✅ 自己达到筑基期或更高
- ✅ 散布谣言至少5次（削弱25%战力）
- ✅ 威胁度低于50%

点击"发起生死决斗"，查看战力对比：
- 你的战力 > 敌方战力 → 胜率高
- 确认后进入最终战斗

### 6. 完成复仇

**胜利**:
- 宿敌死亡
- 邸报头条："楚家家主暴毙！神秘大能复仇成功！"
- 解锁复仇结局

**失败**:
- 重伤，HP和修为大损
- 但你还活着，可以继续修炼后再战

## 💡 新手技巧

### 技巧1: 稳扎稳打
- 前2年专注修炼，不要急着散布谣言
- 等自己达到筑基期再开始复仇计划

### 技巧2: 低成本策略
- 每次只用50灵石散布谣言
- 每散布3次后隐匿1月
- 虽然慢，但安全

### 技巧3: 激进策略
- 每次用100+灵石散布重度谣言
- 快速削弱宿敌
- 威胁度飙升后立即隐匿3月
- 风险高但节省时间

### 技巧4: 看准时机
- 第1年：宿敌筑基，战力飙升
- 第3年：宿敌结婚金丹，战力再升
- 建议在第2年或第4年发起决斗

### 技巧5: 计算战力
- 每次谣言削弱5%战力
- 10次谣言 = 削弱50%
- 配合自身修炼，确保战力超过对方

## ⚠️ 常见错误

### ❌ 错误1: 过早决斗
- 只有练气期就去挑战
- **正确做法**: 至少达到筑基期

### ❌ 错误2: 忽视威胁度
- 威胁度超80%还在散布谣言
- **正确做法**: 立即隐匿行踪

### ❌ 错误3: 不看邸报
- 忽略威胁度预警新闻
- **正确做法**: 每月打开邸报查看

### ❌ 错误4: 资源不足
- 没留足够灵石应急
- **正确做法**: 至少保留200灵石备用

### ❌ 错误5: 时间浪费
- 过度隐匿，浪费修炼时间
- **正确做法**: 威胁度<60时不需要隐匿

## 🎯 三种结局路线

### 路线A: 速通复仇（高风险）
1. 第1年修炼到炼气后期
2. 第2年开始疯狂散布重度谣言（100灵石）
3. 散布5次后立即隐匿
4. 第3年宿敌结婚前发起决斗

**优点**: 快速完成
**缺点**: 风险高，容易被刺杀

### 路线B: 稳健复仇（推荐）
1. 前2年专注修炼到筑基期
2. 第3年开始散布轻度谣言（50灵石）
3. 每散布3次隐匿1月
4. 第5年宿敌生子后发起决斗

**优点**: 安全稳妥
**缺点**: 耗时较长

### 路线C: 极限复仇（终极挑战）
1. 修炼到金丹期
2. 不散布谣言，正面硬刚
3. 用绝对实力碾压

**优点**: 最有成就感
**缺点**: 需要大量时间和资源

## 🎮 实战案例

```
第1年春：
- 查看邸报："楚清瑶一月筑基！"
- 行动：专注修炼，不理会

第2年夏：
- 自己达到炼气后期
- 行动：散布谣言（70灵石）
- 结果：成功，威胁度15%

第2年冬：
- 邸报："楚清瑶订婚"
- 威胁度30%，邸报出现悬赏令

- 结果：威胁度降至0%

第3年春-秋：
- 连续散布谣言4次
- 威胁度累积到55%
- 修为继续提升到筑基初期

第3年冬：
- 邸报："楚清瑶大婚"
- 行动：最后1次重度谣言（100灵石）
- 威胁度飙升到80%

第4年春：
- 邸报："血榜杀手出动！"
- 行动：立即隐匿3月
- 威胁度降至0%

第4年夏：
- 谣言累计6次，削弱30%战力
- 自己筑基中期
- 战力对比：我2000 vs 敌1400
- 行动：发起生死决斗
- 结果：胜利！复仇成功！
```

## 📖 进阶玩法

解锁后可尝试：

1. **最少谣言挑战**: 只散布3次谣言就决斗
2. **零隐匿挑战**: 全程不使用隐匿行踪
3. **速通挑战**: 3年内完成复仇
4. **极限挑战**: 炼气期决斗胜利
