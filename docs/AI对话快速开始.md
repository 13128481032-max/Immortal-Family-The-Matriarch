# AI 对话功能 - 快速开始 ⚡

> 3 分钟配置，立即体验 NPC 智能对话！

## 🎯 第一步：获取 API Key（推荐 DeepSeek）

### 为什么选择 DeepSeek？
- ✅ **便宜**：¥10 可以聊几千次（约 0.03 分/次）
- ✅ **中文强**：专门优化过中文角色扮演
- ✅ **快速**：2-5 秒响应
- ✅ **国内**：无需科学上网

### 获取步骤
1. 访问：https://platform.deepseek.com/
2. 注册/登录账号
3. 进入【API Keys】页面
4. 点击【创建新密钥】
5. 复制你的 Key（格式：`sk-xxxxxx`）

## 🔧 第二步：配置游戏

1. **启动游戏**
2. 点击底部导航栏的【⚙️ 系统】
3. 找到【🤖 AI 对话配置】卡片
4. 填写以下信息：

```
API Key: [粘贴你的 sk-xxxxx]
API URL: https://api.deepseek.com/chat/completions
模型名称: deepseek-chat
```

5. 点击【🔌 测试连接】（等待 2-3 秒）
6. 看到 ✅ 成功提示后，点击【💾 保存配置】

## 💬 第三步：开始对话

1. 点击任意 NPC 卡片，打开详情页
2. 点击顶部的【💬 传音对话】标签
3. 在输入框输入消息，按 Enter 发送
4. 享受智能对话！

## 💡 进阶技巧

- **记忆延续**：聊天记录自动保存，下次继续
- **清空记录**：点击右上角【清空】按钮
- **影响因素**：好感度、性格、身份、境界都会影响对话

## ❓ 常见问题

**Q: 没有 DeepSeek 账号怎么办？**  
A: 也可以用 OpenAI（需科学上网）或通义千问（阿里云）。

**Q: AI 回复太慢？**  
A: 正常情况 2-5 秒，耐心等待即可。

**Q: 配置错误怎么办？**  
A: 点击【测试连接】查看错误提示，通常是 Key 或 URL 填错了。

**Q: 多少钱？**  
A: DeepSeek 约 ¥10 充值可聊 3000+ 次，几乎免费。
