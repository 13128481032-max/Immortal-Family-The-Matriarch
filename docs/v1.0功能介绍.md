# 嫡女仙途 v1.0 功能介绍

> 一款基于 React 的修仙养成游戏，融合剧情、养成、探险等多种玩法

---

## 🎮 核心玩法

### 👤 角色扮演
- 扮演楚家嫡女「楚清辞」，经历从退婚到复仇的仙途之路
- 通过修炼提升境界，从炼气初期到元婴老祖
- 多维度角色成长：修为、资质、灵根、战斗力

### 💑 情缘系统
**NPC攻略**：
- 🔍 **外出游历**：偶遇不同身份、资质的攻略对象
- 💬 **闲聊互动**：通过对话增进好感度
- 🎁 **赠礼系统**：从背包选择丹药、武器等物品赠送（v1.0 新）
- ⚔️ **切磋战斗**：友好比试提升修为
- 🧘 **双修系统**：达到80+好感后共同修炼
- 👶 **劝生功能**：说服对方诞下子嗣

**NPC特性**：
- 20种身份（剑修、炼丹师、佛修、魔修等）
- 20种性格（温柔、高冷、病娇、忠犬等）
- 灵根系统（天灵根、单灵根、双灵根等）
- 宗门归属（凌霄剑宗、丹鼎宗、金刚寺等20个宗门）
- 修为自动增长（资质、灵根、好感度影响速度）
- 寿元系统（境界突破延长寿命）

### 👨‍👩‍👧 子嗣养成
**繁衍系统**：
- 👶 生育子女，遗传父母属性
- 🧬 灵根遗传（优质灵根概率更高）
- 📊 资质培养（教导提升资质）
- 🎓 宗门选择（12岁测灵后加入宗门）
- ⚡ 修为反哺（子女修为提升反馈给主角）

**子嗣管理**：
- 📖 功法系统（不同品级功法影响修炼速度）
- 🎒 装备系统（武器、防具、饰品）
- 💊 物品使用（丹药、灵乳等提升修为/资质）
- 📊 修为进度可视化
- 👥 家族树状图展示

### 🏪 资源系统
**修炼界面**：
- 🧘 **修炼**：每月提升修为经验
- 💼 **打工**：赚取灵石
- 🛒 **下山采购**：购买草药、丹药、武器、功法（v1.0 新）
  - 随机生成3-5个商品
  - 价格根据物品等级浮动
  - 高境界玩家遇到稀有物品概率更高

**背包系统**：
- 📦 存储获得的所有物品
- 🎁 物品可赠送给攻略对象或子女
- 🔧 装备可装配到子女身上
- 💊 消耗品可使用提升属性

### 🗺️ 探险系统
**秘境挑战**：
- 🏔️ 多个不同难度秘境（迷雾森林、火焰山、寒冰窟等）
- 🎲 随机事件（战斗、宝箱、NPC遭遇、选择）
- ⚔️ Boss战斗
- 💎 丰厚奖励（功法、武器、丹药）
- 👥 子女组队探险

### 🎭 剧情系统 v2.0
**多样化剧情**：
- 💕 **恋爱剧情**：20条根据性格分类的浪漫故事
- 👨‍👩‍👧 **亲子剧情**：15条温馨家庭故事（按年龄段）
- 🎯 **身份剧情**：21条独特职业故事

**触发机制**：
- 通过「闲聊」按钮触发
- 根据好感度、关系阶段、子女年龄自动匹配
- 智能概率调控避免重复

### 🤖 AI对话系统（可选）
**智能对话**：
- 💬 与NPC进行自然语言对话
- 🎭 根据性格、好感度动态调整对话风格
- 🏛️ 记忆系统：NPC记住重要事件
- 👶 游戏状态集成：AI知晓你们的孩子、关系等
- 🔮 支持多种AI服务（DeepSeek、OpenAI等）

### 📖 NPC日志系统
**第一人称日志**：
- 📝 NPC以第一人称记录与玩家的互动
- 💬 互动日志（闲聊、赠礼、双修等）
- ⚡ 大事日志（突破境界、生子等）
- 📖 日常日志（修炼、思念等）
- 🔒 高好感度解锁私密日志

### 💰 家族经营
**产业系统**：
- 🏪 购买产业（客栈、丹房、灵田等）
- 👨‍💼 任命子女为掌柜
- 💵 每月获得收益
- 📈 管理者资质影响收益

### ⚔️ 复仇系统
**对抗宿敌**：
- 🗡️ 散布谣言（降低对方修为和名望）
- 🛡️ 隐匿行踪（降低威胁度）
- ⚔️ 生死决斗（最终对决）
- 📊 实时追踪宿敌修为和威胁度

## 💡 游戏提示

### 修炼建议
- 资质越高，修炼速度越快
- 灵根品质决定修炼上限
- 功法品级影响修炼效率
- 宗门加成提供额外速度

### 攻略建议
- 不同性格NPC需要不同应对方式
- 投其所好的礼物效果更佳
- 高好感度解锁更多剧情
- 切磋和双修可以共同提升修为

### 子女培养
- 优先提升资质（影响终身）
- 选择合适的功法很重要
- 12岁前无法加入宗门
- 装备可以显著提升战力

### 资源管理
- 灵石用途广泛，需谨慎使用
- 秘境探险收益高但有风险
- 产业是稳定的收入来源
- 商店物品每次刷新都不同
