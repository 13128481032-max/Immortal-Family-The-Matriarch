# 新手引导弹窗 - 快速开始

## 🎯 功能简介

为《嫡女仙途》游戏添加了一个精美的5步新手引导弹窗，帮助新玩家快速上手游戏。

## 🚀 如何触发

### 自动触发（推荐）
新玩家首次进入游戏时会自动显示引导弹窗：
1. 启动游戏
2. 完成序章（角色创建）
3. 进入主游戏界面
4. 🎉 引导弹窗自动弹出

### 手动触发（测试用）
如果你已经完成过引导想再次查看：

**方法1: 浏览器控制台**
```javascript
localStorage.removeItem('has_read_tutorial_v2');
location.reload();
```

**方法2: 开发者工具**
1. 按 F12 打开开发者工具
2. 切换到 "Application" 或 "应用" 标签
3. 找到 "Local Storage" → 你的域名
4. 删除 `has_read_tutorial_v2` 这一项
5. 刷新页面

## 📖 引导内容

引导分为5个步骤，每步都有清晰的说明：

### 步骤1️⃣: 欢迎来到《嫡女仙途》
- 游戏背景介绍
- 复仇目标说明
- 核心玩法概览

### 步骤2️⃣: 游戏界面导航
- 顶部状态栏详解
- 底部导航栏介绍
- 各个功能面板说明

### 步骤3️⃣: 如何快速提升修为
- 修为反哺机制深度解析
- 境界系数影响
- 策略建议和技巧

### 步骤4️⃣: 注意威胁度
- 威胁度系统说明
- 刺杀触发条件
- 战斗系统介绍

### 步骤5️⃣: 开始你的仙途吧
- 小提示总结
- 完整指南查看方式
- 鼓励和祝福

## 🎮 操作方式

### 基础操作
- **下一步**: 点击右下角蓝色/金色按钮
- **上一步**: 点击左下角灰色按钮（第2步开始出现）
- **跳过引导**: 点击右上角 "跳过引导" 按钮
- **关闭弹窗**: 完成所有步骤后点击 "开始游戏"

### 进度显示
- 顶部有实时进度条
- 左上角显示当前步骤 (如: 步骤 2 / 5)

## 💡 实用技巧

### 对于新玩家
1. 建议完整看完所有5步引导
2. 特别注意 **修为反哺机制** 的说明（最重要！）
3. 完成引导后可随时在【系统】→【仙途指南】查看详细攻略

### 对于老玩家
- 引导只会在首次进入时显示
- 不会打扰你的正常游戏流程
- 如需回顾，可查看系统面板中的详细指南

### 对于开发者
- 修改引导内容：编辑 `src/components/Modals/TutorialModal.jsx`
- 调整触发逻辑：修改 `src/App.jsx` 中的 useEffect
- 重置测试环境：运行 `scripts/testTutorial.js` 中的函数

## 🆘 常见问题

### Q1: 引导弹窗没有出现？
**A**: 可能是因为：
- 你已经完成过引导（localStorage中有标记）
- 游戏还在序章阶段，还没进入主游戏
- 解决方法：执行上面的"手动触发"步骤

### Q2: 如何再次查看引导？
**A**: 
- 快速方法：在浏览器控制台执行
  ```javascript
  localStorage.removeItem('has_read_tutorial_v2');
  location.reload();
  ```
- 或者查看【系统】面板中的【仙途指南】（更详细）

### Q3: 跳过引导后还能看吗？
**A**: 可以！有两种方式：
1. 手动触发引导（见上文）
2. 查看系统面板中的详细指南

### Q4: 引导内容会更新吗？
**A**: 会的！每次游戏版本更新可能会优化引导内容。如果更新了引导内容，会使用新的版本号（如 v3, v4），你会再次看到新引导。

### Q5: 引导占用存储空间吗？
**A**: 几乎不占用。只在 localStorage 中保存一个小标记（约20字节）。
